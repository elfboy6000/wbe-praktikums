Vier gewinnt — Mini Project Documentation

Team

  • Michael Yorke Dobeson

Features

  • Playable Connect-4 board (6×7)
  • Turn handling: red/blue alternating
  • Winner detection (4 in a row) + draw detection
  • Multiple Undo (step-by-step back to start)
  • Persistence using LocalStorage (Save / Load)
  • UI built with React + SJDON (no build step)

How to run

  • Open c4.html via a local server (recommended) or GitHub Pages.

Notes

  • Undo works because state updates are immutable and old states are stored in a stack.
  • Save/Load stores both current state and undo history.
